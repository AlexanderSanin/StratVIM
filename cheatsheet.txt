## nvim-tree
Open directory tree | <C-f>

## toggleterm
Open terminal in floating window | <C-t>

## markdown-preview
Get preview of the markdown file | :MarkdownPreview

## LazyGit
Open LazyGit in current git directory | <C-l>

## CheatSheet
Open CheatSheet | <C-h>

## Comment
Toggle comment on a line | gcc
Toggle comment on a block | gbc
Comment out line | gc
Comment out block | gb
